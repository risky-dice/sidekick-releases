# 예산표 체크리스트

예산표가 포함된 문서에만 선택 적용한다.

## 금액 검증
- 예산 총액이 공문 기준 금액과 일치하는지 확인한다.
- 각 행의 금액 합계가 총액과 일치하는지 계산한다.
- 산출 내역의 단가와 수량 곱이 금액과 맞는지 확인한다.

## 제한 비율
- 자산취득비 제한이 있는 경우 총액 대비 비율을 계산한다.
- 업무추진비 제한이 있는 경우 총액 대비 비율을 계산한다.
- 제한 예외 조항이 있으면 문서 근거를 함께 확인한다.

## 항목 분류
- 노트북, 데스크탑, 완제품 장비 등 반복 사용 자산은 자산취득비 여부를 검토한다.
- 키트, 소모품, 부품, 제작 재료는 일반운영비/재료비로 분류 가능한지 검토한다.
- 구독료는 사용 기간, 서비스명, 수업 활용 목적이 드러나게 쓴다.

## 제출 전
- 예산집행분야, 집행 내용, 금액, 산출 내역, 비고가 모두 채워졌는지 확인한다.
- 표 안의 줄바꿈이 읽기 좋게 들어갔는지 확인한다.
- 금액 표기가 `원` 단위로 통일되었는지 확인한다.
