# 공모/운영계획서 체크리스트

공모 신청서, 운영계획서, 사업계획서에 적용한다.

## 기본 정보
- 사업명이 원문 공문과 일치하는지 확인한다.
- 학교명, 운영 기간, 대상 학년/학과가 일관되는지 확인한다.
- 서식 번호와 제목이 양식과 맞는지 확인한다.

## 내용 구성
- 운영 목적, 운영 여건, 운영 계획, 특색 사업, 지역 연계, 기대효과가 빠지지 않았는지 확인한다.
- 평가 기준이 있는 경우 추진 역량, 운영 계획, 운영 과제, 운영 결과, 예산 집행이 드러나게 작성한다.
- 학교 현황과 실적은 실제 학교 사업명과 연도에 맞춘다.
- 학생 수, 학급 수 등 통계는 최신 기준으로 확인한다.

## 제출 전
- 서식1, 서식2 등 요구 서식이 모두 포함되었는지 확인한다.
- 신청교 현황 파일이 별도 붙임이면 누락하지 않는다.
- 문서 안의 표, 쪽수, 제목 체계가 흐트러지지 않았는지 확인한다.
