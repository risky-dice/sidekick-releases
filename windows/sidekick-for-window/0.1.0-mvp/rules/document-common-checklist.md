# 문서 공통 체크리스트

모든 문서 작업에 기본 적용한다.

## 작업 전
- 원본 파일 위치를 확인한다.
- 같은 이름의 백업 파일을 먼저 만든다.
- 백업 파일명은 `원본파일명.backup-before-작업명-YYYYMMDD.ext` 형식으로 둔다.
- 최종본, 작업본, 백업본이 섞이지 않도록 저장 위치를 정한다.

## 수정 중
- 사용자가 요청한 범위 밖의 문단, 표, 서식은 건드리지 않는다.
- 표 안의 내용은 셀 경계를 유지한다.
- 한글, 특수문자, 금액, 날짜가 깨지지 않는지 확인한다.
- 숫자와 합계가 바뀌는 경우 반드시 다시 계산한다.

## 수정 후
- 최종 파일이 실제로 열리는지 확인한다.
- 수정한 부분을 다시 추출하거나 눈으로 확인한다.
- 제출용 파일명에 학교명, 사업명, 문서 종류가 들어가는지 확인한다.
- 불필요한 임시 파일은 최종본과 분리한다.
