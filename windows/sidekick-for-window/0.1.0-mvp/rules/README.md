# Sidekick Rules

문서 작업을 시작할 때 필요한 체크리스트만 골라서 적용한다.

## 기본 적용
- `document-common-checklist.md`: 모든 문서 공통
- `hwp-safety-checklist.md`: HWP/HWPX 문서를 직접 수정할 때

## 문서 유형별 선택 적용
- `official-letter-checklist.md`: K-에듀파인 공문 기안/발송
- `application-plan-checklist.md`: 공모 신청서, 운영계획서, 사업계획서
- `budget-checklist.md`: 예산표가 포함된 문서에만 적용

## 작업 폴더
- `backups`: 원본 백업 보관
- `logs`: 수정/검증 기록 보관
- `tools`: 자동화 스크립트 보관
