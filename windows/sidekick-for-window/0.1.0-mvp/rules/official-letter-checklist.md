# 공문 기안/발송 체크리스트

K-에듀파인 공문을 작성하거나 제출할 때 적용한다.

## 본문
- 제목에 사업명과 제출 목적이 드러나는지 확인한다.
- 관련 문서번호와 날짜가 정확한지 확인한다.
- 본문은 공문서 문체로 간결하게 작성한다.
- 제출 기한과 수신처가 원문 공문과 일치하는지 확인한다.

## 붙임
- 붙임 번호와 실제 첨부파일 순서가 일치하는지 확인한다.
- 붙임 파일명에 학교명과 문서 종류를 넣는다.
- 운영계획서, 신청교 현황, 기타 요구 서류가 누락되지 않았는지 확인한다.

## 발송 전
- 수신처가 정확한지 확인한다.
- 마감 시간이 임박한 경우 발송 완료 화면 또는 문서번호를 캡처한다.
- 공문 요청 시 발송 예정인 자료는 별도로 표시한다.
